1. Introduction of CloudDrive

Are you looking for a reliable, convenient and free online storage solution? CloudDrive is the your choice!  CloudDrive is a client software for Adobe Acrobat Storage Service, which offers 5GB free online storage for each account.So, when you use CloudDrive, all your files are stored in Adobe Acrobat Storage Service.

Why we need such a client software? Cann't I use Acrobat storage service directly? Ok, here is the reason: Adobe Acrobat File Storage Service, like most other online storage solutions, such as Google Docs, Microsoft Skydrive, only offers you to upload/download files through their websites. Thinking about uploading a large file or folder, which probably takes hours to finish, in the whole process, you cannot close the webpages or turn off your computer! There is no way to do the lengthy job silently in backgourd. If this is not too bad? Here is the worst thing: All these services do NOT allow you to upload a folder! Want to upload a folder with hundreds of files/subfolders inside of it? Forget it! It would be a nightmare!

With CloudDrive, you can upload/download a folder and all the files/subfolders inside of it with just a drag and drop! CloudDrive will take care of all the uploading/downloading in the background.

With CloudDrive, you do not need to worry anymore about losing your data due to computer hard disk failures or outsider threats. You can be sure your important files will always be there when you need them.

With CloudDrive, you can access to your data from anywhere you are connected to the Internet.


2. Quick Start Guide

2.1  Login

When you start up CloudDrive, it will ask you to input your registered email and password on Adobe Acrobat Storage Service to login.

If you do not have an account on acrobat.com, click the “sign up” link or visit Acrobat.com to sign up for a free account. The sign up page is very simple and friendly. Only your email, password, name and country are required.

After that, your registered mailbox will receive an email from acrobat. You MUST confirm your email by clicking the link in the email.

Now input your registered email and password in the login form and start to experience CloudDrive. When the first time you logged in, probably there are already several files in CloudDrive;  All of them are Acrobat’s ads, you can delete them all.

2.2  Upload files

There are 3 ways to upload files or folders:

a)  The most convenient way is: Drag the uploading  files and folders in windows explorer, and drop them in the CloudDrive window. Done! Is it easy?

b) Copy&Paste: Select the files and folders in windows explorer, right click, select “copy” in the context menu (or Ctrl+C). Go back to CloudDrive window, right click, “Paste” (or Ctrl+V). 

c) Click the “upload” button in the toolbar, select files in the popup dialogbox, then click “open” to start uploading.    NOTE: you can not upload folders through this way.

2.3  Download files

Select files and folders in CloudDrive window, right click, select “Download…”, pick up a folder in the popup dialog, click “OK” to start download.

NOTE: Drag files/folders from CloudDrive to windows explorer is not supported yet. This feature will be included in the future version of CloudDrive.

2.4  Other functions

Double Click a file will start to download the file to local computer and open it.

Double Click a folder will get into the folder.

Click the "Parent Folder" button on the toolbar will go back to the upper level folder.

Delete and Rename are supported.

3. F.A.Q

1. Is the online storage free?

Yes, Acrobat offers 5GB free storage for each registered account.

2. Is CloudDrive software free?

Yes, this software is completely free.

3. Is it safe to store my files online?

Yes. You can trust Adobe Acrobat more than your hard disk. With CloudDrive, all network traffic are SSL encrypted, so you do not need to worry about data interception.

4. Is there any limitation for the file to be uploaded?

Files larger than 50MB sometime cannot be uploaded successfully, though Acrobat declares there is no problem to upload any file less than 2GB.  We will find out the reason and solve the problem in the future version.

Due to the limitation of Acrobat, there are several formats are not supported yet, like MP3 files. Future version of CloudDrive will support any file format.

5. 5GB storage space is not enough for me, can I have a larger space to store my files?

There is a workaround: you can register more than one account on Acrobat.com, each will have a 5GB free storage apace. So the total space you have is : 5GB * the number of your account.

6. Why I can’t download files or folders by drag them from CloudDrive to my Desktop?

Due to some reason,this hasn’t been supported yet. It will be added in Version 1.0.

7. How can I upload a bunch of files and folders?

Drag them from windows explorer and drop them on CloudDrive.

8. Is there a requirement for my computer?

Till now, CloudDrive can only run on windows XP, windows Vista and windows 7. Some mobile platforms will be supported in the future.

9. Is there a support service for CloudDrive?

There is no official support service for CloudDrive, now. However, if you have any suggestion, bug report or problem of using CloudDrive, please email us. We are glad to hear any feedback from our users.

10. Sometime, CloudDrive is not stable, why?

The current version is 0.68, which means there are still bugs in it.

Another reason is Acrobat storage service has some yet unknown limitations for uploading and downloading. It also perform virus scanning for all uploading files which probably block some requests. And, the complexity of the Internet and your network environment can also lead to occasional loss of data.

We will try to fix all the problems in future verion. And we are thankful to hear bug reports from you.

11. How can I get the latest update for CloudDrive?

You can visit CloudDirve's website: www.driveoncloud.com to get the latest version. Automatic upgrade will be supported in the future.

12. Does the CloudDrive program quit when I close the CloudDrive window?

No, when you close the window of CloudDrive, it is minimized to a system tray icon at the bottom-right corner on your desktop and continues to work in background. The only way to quit the program is to right click the system tray icon and select "exit".

13. What features are going to be added to CloudDrive in the future?

We plan to add File Sharing, Data Encryption, PDF Creation, Windows Explorer Integration and other functions in the future version of CloudDrive.

And we will add more based on users' feedback!
